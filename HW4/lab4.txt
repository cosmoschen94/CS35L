1. First I installed Git by adding it PATH variable.
$ export PATH=/usr/local/cs/bin:$PATH

2. I git cloned Diffutils into a new directory called gitroot.
$ mkdir gitroot
$ git clone git://git.savannah.gnu.org/diffutils.git

3. I entered the repo and got file git-log.txt and git-tags.txt
$ cd diffutils
$ git log > git-log.txt
$ git tag > git-tags.txt

4. I opened git-log.txt using emacs and found the hash value, copied it and  then generated a patch for that commit.
$ emacs git-log.txt
$ M-s w maint: qu
$ git show 62ca21c8c1a5aa3488589dcb191a4ef04ae9ed4f > quote-patch.txt

5. After generating patch, I checked out Diffutils version 3.0
$ git checkout v3.0

6. Then I applied quote-patch.txt to version 3.0 and skipped past some files by typing RET
$ patch -p1 < quote-patch.txt

7. Got an overview of what happened, which showed changes not staged and untracked files.
$ git status
